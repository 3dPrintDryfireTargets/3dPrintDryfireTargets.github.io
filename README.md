# 3D Printing Training Files

This repository hosts a small, self-contained training resource for 3D printing.

It includes a downloadable ZIP file containing printable part files (STLs) and
supporting documentation for creating target stands.

The files are provided freely for educational and personal use. Nothing in the repository is intended or allowed to be repurposed for commercial use.

---

## Contents

The downloadable ZIP includes:

- **STL files** — printable parts for the training exercise
- **Build / usage instructions** (PDF)

All files are packaged together for convenience.
